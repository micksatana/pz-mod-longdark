# Project Zomboid - Long Dark mod

Experiencing rough winter in Project Zomboid

## Features

- Endless snowing
- 50% chance of having a Winter storm daily
- Random min and max temperatures for seasons
  - 0 to -4 °C mininum
  - -20 to -25 °C maximum
